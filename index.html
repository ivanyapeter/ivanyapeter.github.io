ABOUT

A brief background of me

My interest toward web development began when I took a web development task and I realise I enjoy both coding and design. Having background in both software engineering and branding, I have transferrable skills in technical analysis, strategic planning, and user research. I am able to utilize API data to develop web applications and always on the lookout for the best development library and tools to create more engaging web applications.

I am capable of working with the latest web development technologies such as: HTML5, CSS3, Bootstrap, Foundation, JavaScript, Backbone.js, React, Ruby, Ruby on Rails, Node.js, SQL, and I am particularly fond of Restful APIs. I also have experience in front-end frameworks and libraries with APIs such as Google Maps. On top of these technical web development skills and a background in branding, I also develop my projects with the end-user in mind to ensure the experience I would like to create is pleasing for the target user market.
